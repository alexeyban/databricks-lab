# TASK_06: step3_artifact_gen.py — Config & Query Template Generator

## File
`generators/dv_generator/steps/step3_artifact_gen.py`

## Purpose
Takes the classified `DVModel` (hubs + links + satellites from step2) and produces two outputs: (1) the full `dv_model.json` draft config that will ultimately drive the vault notebooks, and (2) per-entity PySpark query snippet files in `query_templates/`. Also scaffolds PIT and Bridge definitions from the vault layer plan and adds them to the model. This is the "paper artifact" step — no notebooks yet (those are step3b).

## Depends on
- `TASK_01: models.py` — `DVModel`, `HubDef`, `LinkDef`, `SatDef`, `PitDef`, `BridgeDef`
- `TASK_03: session.py` — `Session`

## Inputs
- `DVModel` from `02_classification.json`
- `Session` instance

## Outputs
- `{session_dir}/03_dv_model_draft.json` — complete DV 2.0 config (same schema as `pipeline_configs/datavault/dv_model.json`)
- `{session_dir}/query_templates/hub_{name}.py` — one file per hub
- `{session_dir}/query_templates/lnk_{name}.py` — one file per link
- `{session_dir}/query_templates/sat_{name}.py` — one file per satellite
- `{session_dir}/query_templates/_shared_hash.py` — shared SHA-256 hash formula module

## Key classes / functions

```python
import json
from dataclasses import asdict
from pathlib import Path
from ..models import DVModel, HubDef, LinkDef, SatDef, PitDef, BridgeDef
from ..session import Session

# Hardcoded PIT and Bridge definitions from design/dv2/DV2_VAULT_LAYER_PLAN.md
DEFAULT_PIT_TABLES = [
    {"name": "PIT_FILM",     "hub": "HUB_FILM",     "satellites": ["SAT_FILM_CORE", "SAT_FILM_PRICING"], "snapshot_grain": "daily"},
    {"name": "PIT_RENTAL",   "hub": "HUB_RENTAL",   "satellites": ["SAT_RENTAL_STATUS"],   "snapshot_grain": "daily"},
    {"name": "PIT_CUSTOMER", "hub": "HUB_CUSTOMER", "satellites": ["SAT_CUSTOMER_PROFILE"],"snapshot_grain": "daily"},
    {"name": "PIT_PAYMENT",  "hub": "HUB_PAYMENT",  "satellites": ["SAT_PAYMENT_DETAIL"],  "snapshot_grain": "daily"},
]
DEFAULT_BRIDGE_TABLES = [
    {"name": "BRG_RENTAL_FILM", "path": ["HUB_RENTAL","LNK_RENTAL_INVENTORY","HUB_INVENTORY","LNK_INVENTORY_FILM","HUB_FILM"]},
    {"name": "BRG_FILM_CAST",   "path": ["HUB_FILM","LNK_FILM_ACTOR","HUB_ACTOR"]},
]

class ArtifactGenerator:
    """Generates dv_model_draft.json and per-entity PySpark query templates."""

    def __init__(self, model: DVModel, session: Session):
        self.model = model
        self.session = session
        self.output_dir = session.session_dir
        self.templates_dir = session.session_dir / "query_templates"

    def run(self) -> DVModel:
        if self.session.is_step_done("step3_artifact_gen"):
            return self._load_model()

        self._add_pit_and_bridge()
        self.templates_dir.mkdir(exist_ok=True)
        self._write_shared_hash()
        for hub in self.model.hubs:
            self._write_hub_template(hub)
        for lnk in self.model.links:
            self._write_link_template(lnk)
        for sat in self.model.satellites:
            self._write_sat_template(sat)
        self._write_model_json()

        self.session.mark_step_done("step3_artifact_gen")
        return self.model

    def _add_pit_and_bridge(self) -> None:
        """Populate model.pit_tables and model.bridge_tables from defaults."""

    def _write_hub_template(self, hub: HubDef) -> None:
        """Write query_templates/hub_{lower_name}.py with hash key expr + constants."""

    def _write_link_template(self, lnk: LinkDef) -> None:
        """Write query_templates/lnk_{lower_name}.py."""

    def _write_sat_template(self, sat: SatDef) -> None:
        """
        Write query_templates/sat_{lower_name}.py with:
        - DIFF_HASH_EXPR: sha2(concat_ws("||", coalesce(col, "NULL") for each tracked col), 256)
        - HUB_KEY_EXPR: sha2 of hub BK column
        - TRACKED_COLUMNS list
        """

    def _write_shared_hash(self) -> None:
        """Write _shared_hash.py with the canonical hash key and diff hash functions."""

    def _write_model_json(self) -> None:
        """Serialise full DVModel to 03_dv_model_draft.json."""
```

## Logic walkthrough
1. Skip if already done (session gate).
2. `_add_pit_and_bridge()`: inject hardcoded PIT and Bridge definitions from the design plan into `model.pit_tables` and `model.bridge_tables`.
3. Create `query_templates/` subdirectory.
4. Write `_shared_hash.py` with two functions:
   - `hub_hash_expr(bk_col)` → `sha2(concat_ws("||", upper(trim(col(bk_col).cast("string")))), 256)`
   - `diff_hash_expr(tracked_cols)` → `sha2(concat_ws("||", *[coalesce(col(c).cast("string"), lit("NULL")) for c in tracked_cols]), 256)`
5. For each hub/link/sat: render a Python snippet file with all constants needed for the notebook.
6. Serialise model to JSON.

## Acceptance criteria
- `03_dv_model_draft.json` is valid JSON and matches the schema shown above
- `model.pit_tables` has 4 entries; `model.bridge_tables` has 2 entries
- `query_templates/` contains one file per hub (13), link (17), satellite (14) + `_shared_hash.py` = 45 files
- Each hub template file contains `HASH_KEY_EXPR`, `TARGET_TABLE`, `BUSINESS_KEYS`
- Each satellite template file contains `DIFF_HASH_EXPR` and `TRACKED_COLUMNS`
- Importing a template file in Python does not raise errors
